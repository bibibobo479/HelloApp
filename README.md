# Генератор случайных списков

## Описание задачи
Программа генерирует список целых случайных чисел.
- **Длина списка:** N = номер по журналу + 10
- **Диапазон значений:** от 5 до (номер по журналу × 100)

## Среда реализации
- **ОС:** Linux (Ubuntu 22.04)
- **Язык:** Python 3.10+
- **IDE:** Visual Studio Code

## Используемые библиотеки
- `random` (из стандартной библиотеки Python) — для генерации случайных чисел [citation:3]
